Importing contacts from Shippo into Shopify isn't a direct feature, but you can do it manually using CSV files.

  1. Export Contacts from Shippo:

    - Log in to your Shippo account.

    - Navigate to the "Addresses" section.

    - Export the address book to a CSV file.

  2. Format CSV for Shopify:

    - Open the CSV file using a spreadsheet application (e.g., Excel, Google Sheets).

    - Ensure the CSV file matches Shopify's customer CSV format. Required columns include `First Name`, `Last Name`, `Email`, `Phone`, `Company`, `Address1`, `Address2`, `City`, `Province`, `Province Code`, `Country`, `Country Code`, `Zip`, `Accepts Marketing`, etc.

    - Save the formatted file as a CSV.

  3. Import Contacts to Shopify:

    - Log in to your Shopify admin.

    - Go to "Customers."

    - Choose the formatted CSV file.

    - Click "Upload file."

This will import your contacts from Shippo into Shopify.
